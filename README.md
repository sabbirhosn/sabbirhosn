## 🔭 Featured Project — [EyeGlam](https://eyeglambd.com)

A production eyewear e-commerce store I designed, built, and operate end-to-end — live at [eyeglambd.com](https://eyeglambd.com) with real customers and daily orders.

**Highlights**

- Full product catalog with categories, face-shape style guide, cart, and wishlist
- Wholesale portal and custom frames flow for B2B and personalized orders
- Custom admin panel built on Next.js API routes + PostgreSQL (Supabase)
